The Problem

Swaption Pricing Challenge

What is a Swaption?

A financial derivative giving the right to enter an interest rate swap. Pricing depends on a 2D surface: 14 tenors × 16 maturities = 224 prices evolving daily.

The Task

Given ~490 historical rows, predict the next 6 business days (224 features each). Assessment based solely on these 6 predicted rows against hidden ground truth.

Auto-Regressive Approach

Each predicted day feeds back as input for the next:

Window[T-4..T] → PCA → Model → Pred[T+1]
Window[T-3..T, Pred[T+1]] → PCA → Model → Pred[T+2]
… (6 days total)

Why Residual Architecture Works

⚠️
Barren Plateaus

At 12+ modes, VQC gradients vanish exponentially. QRC sidesteps this by freezing all quantum params.

🔧
Residual Architecture

Quantum only corrects the ~5% residual from a strong classical baseline (R²=0.95).

🔊
Noise Resilience

Photon loss creates different but useful features. Classical readout adapts.
